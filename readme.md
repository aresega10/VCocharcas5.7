Vcocharcas POS v5.7 🛒

Sistema de Punto de Venta (POS) ligero, diseñado específicamente para la gestión de inventario, ventas y fiados en dispositivos móviles.

✨ Características Principales

Gestión de Stock: Control de productos con costos, precios y categorías.

Carrito de Ventas: Interfaz rápida para agregar productos y calcular totales.

Sistema de Fiados: Registro de deudas con integración directa a WhatsApp para envío de cuentas.

Tickets de Venta: Generación de tickets optimizados para impresión térmica o PDF.

Modo PWA: Instalable en la pantalla de inicio de Android e iOS para uso como aplicación nativa.

Exportación/Importación: Manejo de datos mediante archivos CSV (Excel).

🚀 Instalación y Despliegue

Este proyecto se ejecuta totalmente en el lado del cliente (Navegador).

Sube los archivos Index.html, manifest.json y el icono al repositorio.

Activa GitHub Pages en la configuración del repositorio.

Abre el enlace generado y selecciona "Agregar a la pantalla de inicio" en tu navegador móvil.

🛠️ Tecnologías

HTML5 / CSS3 (Tailwind CSS)

JavaScript Vanilla (ES6+)

FontAwesome para iconografía.

Cloud para persistencia de datos local.

Desarrollado para la gestión eficiente de negocios locales.
